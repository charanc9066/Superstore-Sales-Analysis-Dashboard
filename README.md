# 📊 Superstore Sales Analysis Using Power BI and SQL

## 📌 Project Overview
This project analyzes the sales data of a fictional superstore to uncover meaningful business insights. The analysis includes data cleaning, exploratory data analysis, and interactive visualization to understand sales performance, customer behavior, and product trends.

The project demonstrates how **Python, SQL, and Power BI** can be combined to transform raw data into actionable insights for business decision-making.

---

## 🎯 Problem Statement
Retail businesses generate large volumes of transactional data, but extracting useful insights from this data can be challenging.

The objective of this project is to:
- Clean and transform raw sales data
- Perform exploratory data analysis
- Identify sales trends and patterns
- Visualize insights through interactive dashboards

These insights help stakeholders make **data-driven decisions and identify growth opportunities**.

---

## 🛠️ Tools & Technologies
- **Python** – Data cleaning and preprocessing  
- **SQL (MySQL)** – Data analysis and querying  
- **Power BI** – Dashboard creation and visualization  
- **Excel / CSV Dataset** – Source dataset  

---

## 🔄 Project Workflow

### 1️⃣ Data Cleaning (Python)
Python was used to clean and transform the raw Excel dataset:
- Handling missing values
- Removing unnecessary columns
- Formatting data types
- Converting Excel data to CSV format

### 2️⃣ Data Analysis (SQL)
The cleaned dataset was imported into a SQL database where queries were used to analyze the data.

Examples of analysis:
- Total sales and profit
- Sales by region
- Sales by product category
- Customer purchasing patterns
- Top-performing products

### 3️⃣ Data Visualization (Power BI)
Power BI was used to create interactive dashboards that provide insights into business performance.

The Power BI report contains **three dashboard pages**:
- 📈 Sales Performance
- 👥 Customer Analysis
- 📦 Product Analysis

These dashboards allow users to explore the data through filters and visualizations.

---

## 📊 Key Dashboard Insights
The dashboard provides insights such as:

- Total Sales and Profit
- Sales by Category and Sub-Category
- Regional Sales Performance
- Customer Purchasing Trends
- Top-Selling Products
- Profit Analysis

These insights help businesses **optimize product strategy and improve sales performance**.



## 🚀 How to Use
1. Clone the repository
2. Import the SQL dataset into your database
3. Run SQL queries for analysis
4. Open the `.pbix` file in **Power BI Desktop**
5. Explore the interactive dashboards
